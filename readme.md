# Enjoei teste - iOS

## Project Installation

### Specifications

- [Xcode](https://developer.apple.com/download/more/) 10.1
- [Cocoapods](https://cocoapods.org/) 1.5.4
- iOS >= 10
- Swift 4.2
- Arquitetura MVC

**Installation**

The `cocoapods` tool will install all libraries that have been defined in the project Podfile.

```bash
$ pod install
```
